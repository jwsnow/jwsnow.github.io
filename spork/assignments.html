  4. Motion

    Create a website that illustrates the use of the animation loop to model motion.

    Turning it: To turn in this project, you should place the website in a folder called motion inside of your class website and link the site from your main class homepage.

    Requirements: The site should contain at least these pages:
    • (1 page) A home page with a title and links to the pages listed below. Each page below should include a Spork canvas on which your image is drawn.
    • (1 page) On this page, you will draw a fuzz ball by modifying the Random Acceleration example. To do so, you will need to:
      • Add a loop(); command to your setup() function.
      • In your draw() function, add a small rotation before you start drawing. Do not save or restore.
      • In your setup() function, you might want to set the color to have some transparency so that you can see more "fuzziness."

      • Place the resulting code in a web page that includes buttons to Stop, Go, and Start Over.
